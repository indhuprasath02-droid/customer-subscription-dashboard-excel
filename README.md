Bank Customer Campaign Analytics Dashboard
Overview
This project analyzes a bank’s telemarketing campaign data to understand customer subscription behavior for term deposits. The goal is to identify which customer segments are more likely to subscribe and evaluate the effectiveness of the bank’s campaign strategy using Microsoft Excel.
The project covers the complete analytics workflow:
•	Data cleaning
•	Helper column creation
•	KPI calculation
•	Pivot table analysis
•	Interactive dashboard design
________________________________________
Business Problem
A bank runs telemarketing campaigns to promote term deposit subscriptions. Management wants to understand which customer groups respond better, which campaign patterns are more effective, and what factors influence subscription conversion.
The raw data alone does not provide these answers directly, so this project transforms the dataset into an Excel dashboard for business decision-making.
________________________________________
Objective
Build an Excel dashboard to monitor campaign performance and customer conversion behavior.
Key goals:
•	Measure total subscribers and conversion rate
•	Compare conversion across job, age, education, and marital segments
•	Analyze campaign performance by month, duration, and previous outcome
•	Evaluate loan profile and balance-based conversion behavior
•	Present business insights through a dashboard

Project Workflow
The project follows this structure:
Raw_Data → Cleaned_Data → KPI_Calculations → Pivot_Table → Dashboard
Sheets used:
•	Raw_Data – original dataset
•	Cleaned_Data – cleaned data + helper columns
•	KPI_Calculations – summary KPIs
•	Pivot_Table / Pivot_Tables – pivot-based analysis
•	Dashboard – final visual reporting layer
